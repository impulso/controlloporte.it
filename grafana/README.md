# Grafana dashboard

## Import

To import the dashboard into your Grafana, download the [portchecker-grafana-dashboard.json](https://github.com/dsgnr/portchecker.io/raw/devel/grafana/portchecker-grafana-dashboard.json) file and import it into your server. Select your Prometheus instance and that should be all.
